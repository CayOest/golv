# golv

A game solver library for C++20.

## Dependencies

At the moment, only `GoogleTest` is a dependency for the unit tests.

## Build

```
cmake -S . -B build [-DCMAKE_BUILD_TYPE=Release] 
cmake --build build
```

## Run tests

`./build/tests/unit/unit`

## Run examples

`./build/examples/<insert example binary here>`
